## ✨ Kenapa Portofolio Ini Menarik?
Website ini bukan sekadar kumpulan kode, tapi sebuah **"kanvas digital"** yang didesain menggunakan pendekatan *Estilo Fluent AI*—mengutamakan tampilan layaknya *software enterprise* masa depan namun tetap terasa ringan dan seru.

💡 **Fitur-Fitur Keren di Dalamnya:**
- 🌗 **Dark & Light Mode Pintar** – Tema otomatis tersimpan dan beralih dengan transisi mulus nan elegan.
- 💬 **Typing Effect** – Animasi teks interaktif seolah-olah sedang diketik di bagian *Hero*.
- 📱 **Mobile-First Design** – Bebas distorsi! Nyaman dipandang dari HP, tablet, maupun layar super lebar.
- 🌊 **Scroll Animations** – Komponen-komponen UI akan muncul dengan gaya elegan saat Anda melakukan *scrolling*.
- ✉️ **Formulir Kontak Fungsional** – Ingin terhubung? Pengunjung bisa langsung berinteraksi dengan mudah.

---

## 🛠️ Apa Saja Teknologi di Baliknya?
Website ini dibuat murni tanpa *framework* berat untuk memaksimalkan fleksibilitas dan kecepatan muat halaman (*loading speed*)!

* 🧱 **Struktur:** HTML5 Semantik (Mudah dibaca mesin pencari / *SEO-friendly*).
* 🎨 **Desain:** Vanilla CSS3 yang ditenagai oleh fitur modern `CSS Variables` untuk sistem perwarnaan dan transisi antar tema.
* ⚡ **Interaktivitas:** Vanilla JavaScript (tanpa jQuery atau plugin pihak ketiga berukuran raksasa) demi menjaga performa seringan bulu.

## 💻 Jalankan Sendiri di Komputer Kamu!
Ingin melihat-lihat atau memodifikasi kode ini? Caranya super gampang:

1. **Clone repositori ini:**
   ```bash
   git clone https://github.com/kemasmalfath/Portofolio_WEBSITE.git
   ```
2. **Buka foldernya:**
   ```bash
   cd Portofolio_WEBSITE
   ```
3. **Jalankan:** 
   Buka VS Code, dan gunakan *extension* seperti **Live Server**, atau cukup klik dua kali pada file `index.html` dan nikmati di browsermu!
